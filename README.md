## English

An elegant, modern personal website navigation management system with **three deployment modes**:

- **localStorage Mode**: Pure frontend, data in browser (default)
- **GitHub Gist Mode**: Pure frontend, cross-device sync via Gist ⭐ **Recommended**
- **Backend Mode**: Full stack with MySQL, multi-user support

### ✨ Features

- 🎨 **Modern UI** - React 19 + Material UI 7.0 + Tailwind CSS 4.1
- 📱 **Responsive Design** - Desktop and mobile support
- 🌙 **Dark Mode** - Light and dark theme switching
- 🔄 **Drag & Drop** - DND Kit for groups and sites sorting
- 📊 **Group Management** - Flexible groups with public/private settings
- 💾 **Data Export/Import** - JSON format backup and restore
- 🔐 **Admin Authentication** - Login required for editing (configurable)
- 🔍 **Global Search** - Quick search with `⌘K` / `Ctrl+K`
- 📂 **Sidebar Navigation** - Collapsible group categories
